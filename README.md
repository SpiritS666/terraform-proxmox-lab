# Лабораторная работа: Использование Terraform в Proxmox
## Описание
Конфигурация Terraform для автоматического создания виртуальных машин в Proxmox VE.
## Структура проекта
- provider.tf - настройки провайдера Terraform для Proxmox
- variables.tf - переменные конфигурации
- main.tf - описание виртуальной машины
- .gitignore - список игнорируемых файлов
